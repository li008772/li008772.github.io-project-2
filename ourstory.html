Team member 2: Ryan Jasperson

Hi, my name is Ryan Jasperson and I'm a senior here at St. Thomas. I'm looking to graduate this spring with a degree in Computer Science and a minor in Statistics.

As Wangfan explained above, the project was developed mostly using vanilla JavaScript/HTML/CSS. Some jQuery was used for AJAX as well as editting HTML elements. We were able to use the Leaflet API to make a clean heatmap within the application's map. Along with the OpenAQ API and Leaflet API, we also used a Haversine formula derived from Rosetta Code in order to seemlessly calculate the distance from point A to point B on a sphere. By combining the OpenAQ API functionality with the Haversine formula, it was much easier to get accurate results in terms of distances and data.
